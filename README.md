**사용설명**
```
 - bit선택과 피승수, 승수 입력은 필수적으로 이루어진 후 [실행]버튼을 클릭한다.
 - 선택한 bit로 표현 가능한 범위 밖의 입력은 주어지지 않는다고 가정한다.
 - 재연산 희망하는 경우, 종료 후 재실행한다.
```

**기능추가(예정) 리스트**
```
 - 4bit 간격으로 공백을 주고 싶은데, 심심할 때 추가할 것.
 - 최종 출력 결과값 Decimal에서 3자리 단위로 ','를 추가할 것.
```
